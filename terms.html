1. Общие положения

Framesearch - это локальное веб-приложение для управления личной коллекцией видеоконтента. Используя данное приложение, вы соглашаетесь с настоящими условиями использования.

2. Описание сервиса

Framesearch предоставляет инструменты для:

  • Организации личной коллекции видеоконтента
  • Хранения ссылок на видеобалансеры (iframe)
  • Создания коллекций и категорий
  • Экспорта и импорта данных
  • Обмена кодами доступа к контенту

3. Локальное хранение данных

Все данные хранятся локально в браузере пользователя с использованием технологии IndexedDB. Приложение не передает данные на внешние серверы. Пользователь несет полную ответственность за сохранность своих данных и рекомендуется регулярно создавать резервные копии через функцию экспорта.

4. Ответственность пользователя

Пользователь обязуется:

  • Использовать приложение только в законных целях
  • Не нарушать авторские права при добавлении контента
  • Не использовать приложение для распространения незаконного контента
  • Соблюдать законодательство Российской Федерации

5. Ограничение ответственности

Все права на программное обеспечение Framesearch принадлежат его разработчикам. Приложение распространяется с открытым исходным кодом.
