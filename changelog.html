v4.5.0

MINOR
22 февраля 2026

Новые функции

  • Защита паролем: Добавлена возможность установить пароль на приложение с PBKDF2 хешированием (100,000 итераций)
  • Поддержка музыки: Интеграция с 6 музыкальными сервисами (Spotify, Яндекс.Музыка, SoundCloud, VK Музыка, Apple Music, Deezer)
  • Режимы просмотра: Три режима отображения контента (список, плитка, постеры)
  • Плавающее меню: Единый блок быстрых действий в стиле Material Design Speed Dial
  • Креативный прелоадер: Анимированная кинопленка при загрузке страниц

Улучшения

  • Локальные ресурсы: Lucide Icons и шрифты Roboto теперь загружаются локально (без CDN)
  • Офлайн режим: Полная работа приложения без интернета после первой загрузки
  • Темы: Исправлено мерцание элементов при смене темы и перезагрузке
  • Локализация: Полная поддержка русского и английского языков
  • Система логирования: Красивый вывод в консоль с категориями и эмодзи

Исправления

  • Исправлена проблема с отображением иконок после перезагрузки
  • Исправлена работа кнопки "В избранное" на странице видео
  • Исправлено присвоение типа контента для всех источников
  • Исправлена маршрутизация URL без расширения .html
  • Исправлено сохранение защиты паролем после обновления PWA

Безопасность

  • Реализовано криптографически стойкое хеширование паролей (PBKDF2 + SHA-256)
  • Добавлен таймаут сессии (30 минут неактивности)
  • Генерация случайной соли для каждого пароля
