# Framesearch - Быстрый старт

## Горячие клавиши ⌨️

Framesearch поддерживает следующие горячие клавиши для быстрой работы:

- **Ctrl+K** (или Cmd+K на Mac) - Открыть поиск и сфокусироваться на поисковой строке
- **Ctrl+N** (или Cmd+N на Mac) - Открыть модальное окно добавления нового видео
- **Esc** - Закрыть любое открытое модальное окно

## Типы источников видео 🎬

Framesearch поддерживает 4 типа источников видео:

### 1. Видеобалансеры
- **Iframe URL**: Kodik, Collaps, Alloha, HDRezka

### 2. Прямые ссылки
- MP4, WebM, OGG файлы
- Прямые ссылки на видеофайлы

### 3. Социальные сети
- **YouTube**: Поддержка Video ID или полного URL
- **VK Видео**: Ссылки на видео VK
- **RuTube**: Video ID или полный URL

### 4. Кастомный iframe
- Любой HTML код iframe для встраивания видео

## Темы оформления 🎨

- **6 готовых тем**: Темная элегантность, Фиолетовая ночь, Океанская свежесть, Закатное сияние, Лесная тишина, Розовая мечта
- **Полная кастомизация**: Настройка всех цветов, скругления углов, эффектов наведения
- **Автосохранение**: Тема сохраняется автоматически и применяется при загрузке

## PWA возможности 📱

- Установка на устройство
- Работа в оффлайн режиме
- Автоматические обновления
- Кэширование для быстрой загрузки

## Основные функции

1. **Добавление видео**: Кнопка "+" или Ctrl+N
2. **Создание коллекций**: Организация видео в тематические папки
3. **Поиск**: Ctrl+K для быстрого доступа к поиску
4. **Импорт/Экспорт**: Обмен коллекциями через файлы или коды
5. **Избранное**: Отметка любимых фильмов звездочкой
6. **Фильтры**: Все, Фильмы, Сериалы, Избранное

## Конфиденциальность 🔒

- Все данные хранятся локально в IndexedDB браузера
- Никакие данные не отправляются на серверы
- Опциональное шифрование при экспорте
- Управление cookies и разрешениями
