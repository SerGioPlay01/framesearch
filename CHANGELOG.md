# 📝 История изменений

Все значимые изменения в проекте Framesearch документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [5.0.0] - 2026-02-23

### 🎉 Крупное обновление - Финальный релиз

#### Добавлено
- 📊 **Статистика и аналитика**
  - Автоматическое отслеживание просмотров
  - Учет времени просмотра (минимум 5 секунд)
  - Топ-10 самых просматриваемых видео
  - График активности за последние 14 дней
  - Общая статистика: просмотры, уникальные видео, время
  - Статистика за неделю

- 🏷️ **Умная система тегов**
  - Добавление неограниченного количества тегов
  - Автоподсказки при вводе тегов
  - Облако тегов для визуализации
  - Поиск видео по тегам
  - Отображение тегов на карточках видео

- 📝 **Заметки с таймкодами**
  - Добавление текстовых заметок к видео
  - Привязка заметок к конкретному времени
  - Форматирование таймкодов (MM:SS, HH:MM:SS)
  - Редактирование и удаление заметок
  - Экспорт заметок в текстовый файл
  - Отображение заметок на странице видео

- 🎭 **Режим слайдшоу**
  - Полноэкранная презентация постеров
  - Автоматическое переключение слайдов
  - Настройка длительности (3-30 секунд)
  - Случайный порядок показа
  - Отображение информации о видео
  - Управление с клавиатуры (→, ←, P, I, Esc)
  - Прогресс-бар и счетчик слайдов

- 🎵 **Поддержка музыкальных источников**
  - Spotify
  - Apple Music
  - Yandex Music
  - Автоопределение типа источника

#### Улучшено
- 🎨 Улучшенный дизайн модальных окон
- 📱 Оптимизация для мобильных устройств
  - Увеличенные области нажатия (44x44px)
  - Тактильная обратная связь (вибрация)
  - Адаптивные кнопки действий
- 🌍 Расширенная поддержка переводов
  - Переводы для всех новых функций
  - Поддержка атрибута data-i18n-title для подсказок
- ⚡ Улучшена производительность
  - Оптимизация загрузки скриптов
  - Улучшенная работа с IndexedDB
- 🎨 Улучшенные стили для всех компонентов
  - Единообразный дизайн
  - Плавные анимации
  - Стеклянные эффекты

#### Исправлено
- 🐛 Исправлен порядок загрузки скриптов
- 🐛 Исправлены проблемы с инициализацией i18n
- 🐛 Исправлена работа cookie consent виджета
- 🐛 Исправлено отображение прелоадера
- 🐛 Исправлены CORS ошибки с внешними изображениями
- 🐛 Исправлена работа поиска по тегам
- 🐛 Исправлено отображение заметок на странице видео
- 🐛 Исправлено форматирование дат и таймкодов

#### Технические изменения
- 📦 Добавлены новые модули:
  - `scripts/statistics.js` - менеджер статистики
  - `scripts/tags.js` - менеджер тегов
  - `scripts/notes.js` - менеджер заметок
  - `scripts/slideshow.js` - менеджер слайдшоу
  - `scripts/music-sources.js` - поддержка музыки
- 🎨 Добавлены стили:
  - `styles/features.css` - стили для новых функций
- 📝 Обновлена документация:
  - Добавлены docs для новых компонентов
  - Обновлен README.md
  - Обновлен CHANGELOG.md

## [1.0.0] - 2026-02-21

### 🎉 Первый релиз

#### Добавлено
- ✨ Локальное хранилище данных с IndexedDB
- 🎨 7 готовых тем оформления + кастомизация
- 🌍 Мультиязычность (русский и английский)
- 📁 Система коллекций для организации видео
- ⭐ Избранное для быстрого доступа
- 🔍 Мгновенный поиск по коллекции
- 🔄 Импорт и экспорт данных
- 🔐 Шифрование данных при экспорте (AES-256)
- 🤝 Обмен видео по коду между пользователями
- 📱 PWA функционал с оффлайн поддержкой
- ⌨️ Горячие клавиши (Ctrl+K, Ctrl+N, Esc)
- 🎯 4 типа источников видео:
  - Балансеры (Kodik, Collaps, Alloha, HDRezka)
  - Прямые ссылки на видеофайлы
  - Соцсети (YouTube, VK, RuTube, Vimeo)
  - Custom iframe код
- 🍪 Управление cookies и конфиденциальностью
- 📄 Полная юридическая документация (RU/EN):
  - Условия использования
  - Политика конфиденциальности
  - Политика cookies
  - Лицензия
  - Отказ от ответственности

#### Темы оформления
- 🌙 Темная элегантность (по умолчанию)
- ✨ Фиолетовая ночь
- 🌊 Океанская свежесть
- 🌅 Закатное сияние
- 🌲 Лесная тишина
- 💖 Розовая мечта
- ⚫ Monochrome

#### Технические особенности
- Vanilla JavaScript (без фреймворков)
- Service Worker для оффлайн работы
- Web Crypto API для шифрования
- Responsive дизайн для всех устройств
- Оптимизация производительности
- Кроссбраузерная совместимость
