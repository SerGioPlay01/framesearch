# 🤝 Руководство по внесению вклада

Спасибо за интерес к проекту Framesearch! Мы рады любому вкладу в развитие проекта.

## 📋 Как внести вклад

### 1. Сообщить об ошибке

Если вы нашли ошибку:
1. Проверьте, нет ли уже похожей проблемы в [Issues](https://github.com/SerGioPlay01/framesearch/issues)
2. Создайте новый Issue с подробным описанием:
   - Шаги для воспроизведения
   - Ожидаемое поведение
   - Фактическое поведение
   - Скриншоты (если применимо)
   - Версия браузера и ОС

### 2. Предложить улучшение

Есть идея для новой функции?
1. Откройте Issue с меткой "enhancement"
2. Опишите предлагаемую функцию
3. Объясните, почему это будет полезно

### 3. Внести код

#### Подготовка

```bash
# Форкните репозиторий
# Клонируйте свой форк
git clone https://github.com/ваш-username/framesearch.git
cd framesearch

# Создайте ветку для изменений
git checkout -b feature/ваша-функция
```

#### Разработка

1. Следуйте существующему стилю кода
2. Добавьте комментарии к сложным участкам
3. Тестируйте изменения в разных браузерах
4. Убедитесь, что PWA функционал работает

#### Отправка изменений

```bash
# Закоммитьте изменения
git add .
git commit -m "feat: описание изменений"

# Отправьте в свой форк
git push origin feature/ваша-функция

# Создайте Pull Request на GitHub
```

## 📝 Стиль кода

### JavaScript
- Используйте ES6+ синтаксис
- Именуйте переменные понятно и описательно
- Добавляйте JSDoc комментарии к функциям
- Избегайте глобальных переменных

### HTML
- Используйте семантические теги
- Добавляйте `data-i18n` для переводимых текстов
- Соблюдайте отступы (2 пробела)

### CSS
- Используйте CSS переменные для цветов
- Группируйте связанные стили
- Добавляйте комментарии к секциям

## 🌍 Переводы

Хотите добавить новый язык?

1. Откройте `scripts/i18n.js`
2. Добавьте новый объект языка в `translations`
3. Переведите все ключи
4. Добавьте флаг в переключатель языков
5. Обновите документацию

## 🧪 Тестирование

Перед отправкой PR протестируйте:
- ✅ Добавление/редактирование/удаление видео
- ✅ Создание коллекций
- ✅ Поиск и фильтры
- ✅ Импорт/экспорт данных
- ✅ Смену тем
- ✅ Переключение языков
- ✅ PWA установку
- ✅ Работу в оффлайн режиме

### Браузеры для тестирования
- Chrome/Edge (последняя версия)
- Firefox (последняя версия)
- Safari (если доступен)
- Мобильные браузеры
