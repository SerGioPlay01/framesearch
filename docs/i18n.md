# i18n - Интернационализация

Легковесная библиотека для многоязычности веб-приложений с поддержкой динамической смены языка.

## 📋 Описание

i18n - это простая и эффективная система интернационализации для JavaScript приложений. Поддерживает множество языков, динамическую смену языка и автоматическое обновление интерфейса.

## ✨ Возможности

- 🌍 Поддержка множества языков
- 🔄 Динамическая смена языка без перезагрузки
- 📝 Автоматическое обновление DOM элементов
- 💾 Сохранение выбранного языка в localStorage
- 🎯 Простой API для получения переводов
- 🔍 Fallback на ключ при отсутствии перевода
- ⚡ Легковесная и быстрая

## 🚀 Установка

Скопируйте файл `scripts/i18n.js` в ваш проект и подключите его:

```html
<script src="scripts/i18n.js"></script>
```

## 📖 Использование

### Базовое использование

```javascript
// Получить перевод
const greeting = i18n.t('welcome.message');
console.log(greeting); // "Добро пожаловать!"

// Сменить язык
i18n.setLanguage('en');

// Получить текущий язык
const currentLang = i18n.getCurrentLanguage();
console.log(currentLang); // "en"

// Получить список доступных языков
const languages = i18n.getAvailableLanguages();
console.log(languages); // ["ru", "en"]
```

### HTML атрибуты

Используйте атрибут `data-i18n` для автоматического перевода:

```html
<!-- Текстовый контент -->
<h1 data-i18n="welcome.title">Заголовок</h1>
<p data-i18n="welcome.description">Описание</p>

<!-- Placeholder -->
<input type="text" placeholder="Поиск" data-i18n="search.placeholder">

<!-- Title атрибут -->
<button title="Закрыть" data-i18n="action.close">×</button>

<!-- Aria-label -->
<button aria-label="Меню" data-i18n="menu.label">☰</button>
```

### Добавление переводов

```javascript
// Добавить новый язык
i18n.addTranslations('de', {
    welcome: {
        title: 'Willkommen',
        message: 'Hallo Welt!'
    },
    action: {
        save: 'Speichern',
        cancel: 'Abbrechen'
    }
});

// Обновить существующие переводы
i18n.addTranslations('ru', {
    newFeature: {
        title: 'Новая функция',
        description: 'Описание новой функции'
    }
});
```

### События

```javascript
// Подписаться на смену языка
window.addEventListener('languageChanged', (e) => {
    console.log('Язык изменен на:', e.detail.language);
    // Обновить динамический контент
    updateDynamicContent();

```

## 📊 API

### Методы

| Метод | Описание | Параметры | Возвращает |
|-------|----------|-----------|------------|
| `t(key)` | Получить перевод | key: string | string |
| `setLanguage(lang)` | Установить язык | lang: string | void |
| `getCurrentLanguage()` | Получить текущий язык | - | string |
| `getAvailableLanguages()` | Получить список языков | - | string[] |
| `addTranslations(lang, translations)` | Добавить переводы | lang: string, translations: object | void |

### Множественное число

```javascript
// Добавить переводы с множественным числом
i18n.addTranslations('ru', {
    items: {
        count: {
            one: '{count} элемент',
            few: '{count} элемента',
            many: '{count} элементов'
        }
    }
});

// Функция для выбора правильной формы
function pluralize(count, key) {
    const forms = i18n.t(key);
    const mod10 = count % 10;
    const mod100 = count % 100;
    
    if (mod10 === 1 && mod100 !== 11) {
        return forms.one.replace('{count}', count);
    } else if (mod10 >= 2 && mod10 <= 4 && (mod100 < 10 || mod100 >= 20)) {
        return forms.few.replace('{count}', count);
    } else {
        return forms.many.replace('{count}', count);
    }
}

console.log(pluralize(1, 'items.count')); // "1 элемент"
console.log(pluralize(2, 'items.count')); // "2 элемента"
console.log(pluralize(5, 'items.count')); // "5 элементов"
```

## 🎨 Кастомизация

### Изменение ключа localStorage

```javascript
// В конструкторе класса
this.storageKey = 'myapp_language'; // вместо 'framesearch_language'
```

### Добавление языка по умолчанию

```javascript
// В конструкторе класса
this.defaultLanguage = 'en'; // вместо 'ru'
```

### Кастомная логика обновления

```javascript
// Переопределить метод updatePageTranslations
i18n.updatePageTranslations = function() {
    // Ваша логика обновления
    document.querySelectorAll('[data-i18n]').forEach(element => {
        const key = element.getAttribute('data-i18n');
        const translation = this.t(key);
        
        // Кастомная логика применения перевода
        if (element.tagName === 'INPUT') {
            element.placeholder = translation;
        } else {
            element.textContent = translation;
        }
    });
};
```
