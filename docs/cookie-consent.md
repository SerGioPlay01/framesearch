# Enhanced Cookie Consent Manager

## Новые эксклюзивные функции

### 1. Автоматическое обнаружение скриптов
- **Реал-тайм сканирование**: Автоматически обнаруживает все скрипты и iframe на странице
- **Категоризация**: Автоматически распределяет скрипты по категориям (необходимые, функциональные, аналитика, внешние)
- **Счетчик скриптов**: Показывает количество обнаруженных скриптов в каждой категории

### 2. Мониторинг в реальном времени
- **MutationObserver**: Отслеживает добавление новых скриптов и iframe в DOM
- **Автоблокировка**: Автоматически блокирует неразрешенные скрипты
- **Периодическое сканирование**: Проверяет наличие несанкционированных cookies каждые 10 секунд

### 3. Расширенный контроль cookies
- **Автоудаление**: Автоматически удаляет cookies, которые не разрешены пользователем
- **Категоризация cookies**: Определяет категорию каждого cookie (аналитика, функциональные, внешние)
- **Защита от отслеживания**: Блокирует установку неразрешенных cookies

### 4. Улучшенный интерфейс
- **Детальный список скриптов**: Раскрывающийся список всех обнаруженных скриптов
- **Счетчики**: Визуальные бейджи с количеством скриптов в каждой категории
- **Блокировка контента**: Красивые плейсхолдеры для заблокированных iframe

### 5. Реестр скриптов
- **Отслеживание**: Ведет реестр всех обнаруженных скриптов
- **Метаданные**: Сохраняет информацию о каждом скрипте (источник, категория, статус блокировки)
- **История**: Логирует все действия с cookies и скриптами

## Технические детали

### Обнаружение скриптов
```javascript
// Автоматически обнаруживает:
- Все <script src="..."> теги
- Все <iframe src="..."> теги
- Глобальные объекты (ym, gtag, ga)
- IndexedDB, localStorage, Service Worker
```

### Категории скриптов
1. **Necessary** (Необходимые): IndexedDB, localStorage, Service Worker
2. **Functional** (Функциональные): theme.js, i18n.js, view-mode.js
3. **Analytics** (Аналитика): Yandex.Metrica, Google Analytics
4. **External** (Внешние): Kodik, Collaps, Alloha, HDRezka, Vibix

### Мониторинг
- **MutationObserver**: Отслеживает изменения DOM
- **Периодическое сканирование**: Каждые 10 секунд
- **Cookie перехват**: Перехватывает установку cookies через document.cookie

## Использование

### Базовое использование
```javascript
// Инициализация автоматическая
cookieConsent.init();
```

### Программное управление
```javascript
// Принять все
cookieConsent.acceptAll();

// Отклонить все
cookieConsent.rejectAll();

// Показать настройки
cookieConsent.showSettings();

// Обновить предпочтения
cookieConsent.updatePreference('analytics', false);
```

### События
```javascript
// Слушать изменения языка
window.addEventListener('languageChanged', () => {
    // Обновить интерфейс
});
```

## Стили

Новые CSS классы:
- `.cookie-count-badge` - Бейдж с количеством скриптов
- `.cookie-scripts-details` - Раскрывающийся список скриптов
- `.cookie-scripts-list` - Список обнаруженных скриптов
- `.iframe-blocked-placeholder` - Плейсхолдер для заблокированного контента
