# PWA Icons

## Генерация иконок

Эта папка должна содержать PNG иконки для PWA приложения.

### Требуемые размеры:
- icon-192x192.png
- icon-512x512.png

### Как сгенерировать:

1. **Онлайн сервисы (рекомендуется):**
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/
   - https://favicon.io/favicon-converter/

2. **Загрузите `favicon.svg` из корня проекта**

3. **Скачайте сгенерированные иконки**

4. **Поместите файлы в эту папку**

### Временное решение:

Пока иконки не созданы, приложение использует `favicon.svg` напрямую из manifest.json.

Для полной совместимости со всеми устройствами рекомендуется создать PNG иконки.
