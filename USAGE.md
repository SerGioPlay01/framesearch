# Инструкция по использованию CineGlass

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. При первом запуске автоматически добавятся примеры контента
3. Нажмите на плавающую кнопку "+" чтобы добавить свой контент

## ➕ Как добавить фильм или сериал

### Способ 1: Локальный видео файл

1. Нажмите кнопку "+" в правом нижнем углу
2. Заполните основную информацию:
   - Название
   - Описание
   - Выберите тип (Фильм/Сериал/Аниме)
   - Жанр
   - Год, рейтинг, длительность
3. Загрузите постер (необязательно)
4. Нажмите "Далее"
5. Выберите "Локальный файл"
6. Перетащите видео файл или нажмите "Выбрать файл"
7. Для фильма нажмите "Сохранить"
8. Для сериала нажмите "Далее" и добавьте эпизоды

### Способ 2: Ссылка из видеобалансера

1. Нажмите кнопку "+"
2. Заполните основную информацию
3. Нажмите "Далее"
4. Выберите "Ссылка (iframe)"
5. Вставьте URL из видеобалансера:
   - Kodik: `https://kodik.info/seria/123456/hash/720p`
   - Collaps: `https://collaps.cc/embed/12345`
   - Alloha: `https://alloha.tv/?token=12345`
   - HDRezka: `https://hdrezka.ag/embed/12345`
6. Нажмите "Сохранить"

## 🎬 Добавление эпизодов сериала

1. После заполнения основной информации и источника видео
2. На шаге 3 нажмите "Добавить эпизод"
3. Укажите:
   - Номер сезона
   - Номер эпизода
   - Название эпизода
   - Длительность
4. Выберите источник (файл или ссылка)
5. Нажмите "Сохранить эпизод"
6. Повторите для всех эпизодов
7. Нажмите "Сохранить" для завершения

## 🔍 Поиск контента

### Быстрый поиск
- Используйте поле поиска в навигации
- Или перейдите на страницу поиска
- Нажмите `/` для быстрого доступа к поиску

### Фильтры
- **Тип**: Все / Фильмы / Сериалы / Мультфильмы
- **Жанр**: Выберите из списка
- **Год**: Фильтр по году выпуска
- **Рейтинг**: Минимальный рейтинг (6+, 7+, 8+)
- **Сортировка**: По популярности, рейтингу, дате

### Популярные теги
- Нажмите на тег для быстрого поиска
- Теги обновляются автоматически

## 📺 Просмотр видео

### Локальные файлы
- Встроенный HTML5 плеер
- Полный контроль воспроизведения
- Поддержка горячих клавиш

### Iframe ссылки
- Плеер видеобалансера
- Все функции плеера сервиса
- Выбор качества, озвучки, субтитров

### Управление плеером

**Мышью:**
- Клик на видео - пауза/воспроизведение
- Клик на прогресс-бар - перемотка
- Наведение - показать элементы управления

**Клавиатурой:**
- `Space` - пауза/воспроизведение
- `F` - полноэкранный режим
- `M` - вкл/выкл звук
- `←` - назад на 10 секунд
- `→` - вперёд на 10 секунд

## ⭐ Избранное

1. Наведите на карточку фильма
2. Нажмите на иконку сердца
3. Фильм добавится в избранное
4. Повторное нажатие удалит из избранного

## 🔧 Решение проблем

### Видео не воспроизводится
- **Локальный файл**: Проверьте формат (MP4, WebM, OGG)
- **Iframe**: Проверьте правильность ссылки
- **Браузер**: Используйте современный браузер (Chrome, Firefox, Edge)

### База данных не работает
- Проверьте поддержку IndexedDB в браузере
- Очистите кеш браузера
- Попробуйте режим инкогнито

### Постер не загружается
- Проверьте размер файла (рекомендуется < 1MB)
- Используйте форматы: JPG, PNG, WebP
- Или используйте URL изображения

### Медленная работа
- Ограничьте количество видео (рекомендуется < 1000)
- Используйте сжатые постеры
- Очистите старые данные
